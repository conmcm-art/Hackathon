# Accessibility Layer

This folder centralizes accessibility behavior for Food Bridge.

## Goal
- Target: **WCAG 2.2**
- Practical level: **WCAG 2.2 AA**

## Files
- `accessibility.config.js`: project accessibility targets.
- `focusStyles.js`: shared keyboard focus style classes.
- `reducedMotion.js`: helpers for `prefers-reduced-motion`.
- `ariaLabels.js`: shared ARIA labels.
- `keyboardShortcuts.js`: reusable keyboard shortcut hooks.
